# hs-myserver
hs(Home Server) Project의 첫번째 단계로 다양한 프로젝트를 하나의 환경에서 개발하기 위한 환경 세팅 작업물입니다. On-Premise 쿠버네티스 클러스터에서 동작할 Vscode-server를 구성하는 레포지토리입니다. DockerFile으로 만들어진 개발환경을 Jenkins와 ArgoCD를 통해서 쿠버네티스로 배포합니다.

## 내장 환경
-  Python3
-  Java
-  C
-  C++
-  Node.js
